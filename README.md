# Crypto Trading Simulator
 
A fully functional trading simulator. You should be able to:
- See prices of top currencies
- Buy crypto assets
- Sell crypto assets
- See your holdings
- See your order history

[CoinGecko API](https://www.coingecko.com/en/api#explore-api)
